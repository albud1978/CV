# CV - Computer Vision для гражданской авиации

## Описание проекта

Проект по разработке моделей компьютерного зрения (Computer Vision) для задач гражданской авиации. Основные задачи включают детекцию спецтехники и персонала, контроль соблюдения регламентов и обеспечение безопасности на перроне.

## Структура проекта

```
CV/
├── src/                        # Исходный код
│   ├── configs/                # Конфигурационные файлы
│   ├── inference/              # Скрипты для запуска моделей (инференс)
│   ├── models/                 # Определения архитектур моделей
│   ├── train/                  # Скрипты обучения
│   └── utils/                  # Вспомогательные утилиты
├── input/                      # Входные данные (симлинк на Nextcloud)
├── output/                     # Результаты работы (симлинк на Nextcloud)
├── docs/                       # Документация проекта
├── docker-compose.yml          # Конфигурация Docker Compose
├── Dockerfile                  # Описание Docker-образа
├── README.md                   # Описание проекта
├── CHANGELOG.md                # История изменений
└── requirements.txt            # Python-зависимости
```

## Требования

- **Docker Desktop** (с поддержкой WSL2 на Windows)
- **NVIDIA Driver** (актуальной версии для поддержки GPU)
- Доступ к папкам Nextcloud с данными

## Быстрый старт (Docker)

Рекомендуемый способ запуска — через Docker, что обеспечивает изолированную среду с настроенными драйверами CUDA.

1. **Настройка окружения**
   Создайте файл `.env` в корне проекта и укажите абсолютные пути к папкам данных на вашем компьютере:
   ```bash
   CV_INPUT_PATH="/путь/к/вашей/папке/input"
   CV_OUTPUT_PATH="/путь/к/вашей/папке/output"
   ```

2. **Запуск контейнера**
   ```bash
   docker-compose up -d --build
   ```

3. **Работа внутри контейнера**
   Подключитесь к контейнеру для выполнения команд:
   ```bash
   docker-compose exec cv-dev bash
   ```
   Внутри контейнера доступна карта NVIDIA (проверить: `nvidia-smi`).

## Данные

- **Input**: данные хранятся в Nextcloud: `.../CV/input`
- **Output**: результаты хранятся в Nextcloud: `.../CV/output`
